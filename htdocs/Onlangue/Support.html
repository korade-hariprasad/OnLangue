Introduction :

The OnLangue is an idea to provide a platform for students to practice programming languages.
Here we try to provide maximum language features support without the need of any other plugins or setups.

Cause : The main reason for this is that most of the students don't meet the resources for learning & we believe that for leraning the resources must not be a factor.
This is an try to make these students realize that they are not bound to the resources or anything else.
Only you need is your will to study. If you find the Destination you will find a path :) !!!
Let OnLangue be one of such path for everyone.


Support :

Java
Java is a powerful object oriented language. We provide you the jdk 13.0.1 on Windows Environment.
We tried our best to provide features yet there's room for bugs & hence feel free to report bugs. :)


Python
As most demanding language we thaught we should provide "python" as well. Voila !!! here is the Python v.3.7.4 on windows environment.
Note the python we use are from Anaconda Distribution so there are chances of some minor issues.
Feel free to report bugs. :)


HTML / CSS / Javascript
For now we don't support external css or javascript.
Yet you can use css/javascript by writing them within the HTML page.
Also we cannot risk our servers hence we did not allow the use of external links or local links which direct inside our system.


C / CPP
The Mother of all Languages ! How can we forget that.
We support the C language but this is not the TURBOC Compiler, we use the GCC Project's compiler for c & c++.
To provide a Windows environment we had used MinGW-w64 Project which is targeted to provide compilers for c,cpp,java,fortran,objective c/cpp,ada,D & Go languages.

Instructions :

OK then, let's start with some basics of using this app. Don't worry its a piece of cake.
First create your account. Free of cost, obviously. Then login to your account & follow the following steps.
Don't worry we log you out automatically when you leave the browser.

  1. Select the laguage.
  2. Give a filename
  3. Type the code
  4. Provide parameters if any
  5. Hit Compile


It is important to fill the parameters field with "none" if you don't want to give any parameters.
After this you can see the your code & result.

A list will be provided of your programs below the output. You can download your code file from there.
If by some reasons you cannot download the file then just copy the file link & paste it in other tab's search bar.
You will be able to view the file contents & you can copy them from there.

Language syntax instructions :-

There are no rules for using the java language. It is same as you would do regularly & same is for python.
There is a strict way for syntax of CPP code.Following are some things to remember while working with CPP:
  • No conio.h file
  • No ".h" extension for header files
  • Header files must be written between double quotation marks(" ") & not in between angle brackets(< >).
  • General way :
      #include"stdio"
      int main(){
         //your code here
         return(0);
      }


NOTE : The output depends upon your language. If it is HTML or PHP you will see the web page else you will see the console output.

NOTE : It's a good idea to keep your files backed up as they cannot be restored after deletion.
